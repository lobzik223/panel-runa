# RUNA — Админ-панель

Вход только для аккаунтов, созданных через CLI на сервере (отдельно от пользователей приложения).

## Запуск

1. Установка: `npm install`
2. Логотип: скопируйте `logonoruna.png` в папку `public/` (если ещё не скопирован).
3. Запуск бэкенда: в каталоге `backend-runa` выполните `npm run dev` (порт 3000).
4. Запуск панели: `npm run dev` (порт 5174). Прокси к API: `/api` → `http://localhost:3000`.

## Создание админа (только на сервере, из командной строки)

В каталоге **backend-runa**:

```bash
npm run admin:create -- <email> <пароль> [имя]
# Пример:
npm run admin:create -- admin@runafinance.online "SecurePassword123" "Super Admin"
```

После этого вход в панель по этому email и паролю.

## Разделы

- **DashBoard** — онлайн-пользователи, статистика, графики (обновление каждые 10 мин).
- **Пользователи** — поиск по email/ID, профиль, подписки, блокировки.
- **Документация** — отчёты админов (30 дней).
- **Промокоды** — создание, аналитика, PDF.

Цвета и стиль — в духе приложения RUNA (оранжевый акцент, светлый интерфейс).
